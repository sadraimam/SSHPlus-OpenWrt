# SSHPlus-OpenWrt
SSH Tunnel setup on OpenWrt v22.03.3
Target device: Xiaomi Mi Router 4A Gigabit Edition

## Installation
### Run from ssh
```bash
rm -f /tmp/install.sh && wget -O /tmp/install.sh https://raw.githubusercontent.com/sadraimam/SSHPlus-OpenWrt/refs/heads/main/install.sh && chmod +x /tmp/install.sh && sh /tmp/install.sh
```
